Admin:

Admin is the first user created under the domain, usually with a userid of 1
Admin can login into any room, even with out having to need the permission exclusively defined for the admin user.
Admin will have all Application Privileges





User with Other Privileges:
Application Privileges can allow / disallow a user to do certain actions like
A: Inviting other users
B: Createing rooms
C: Can Access all Rooms
D: Can send Direct Messages
